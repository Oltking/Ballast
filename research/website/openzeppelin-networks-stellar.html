Build Securely on Stellar
with OpenZeppelin

OpenZeppelin brings its trusted smart contracts libraries and developer workflows to Stellar, a Rust-based blockchain. Developers can build payments, stablecoins, and DeFi apps on Stellar using familiar, audited building blocks.

OpenZeppelin Stellar Contracts

A dedicated library of audited contracts and utilities for building on the Stellar blockchain. These standards make it easier to issue assets, launch stablecoins, and create DeFi applications that connect directly to Stellar’s global payment rails.

Docs → Github →

Contracts Wizard

Generate secure Stellar smart contracts quickly and safely. Configure modules, preview the Rust code, and integrate directly into your workflow.

Try Wizard →

Contracts MCP

Integrate OpenZeppelin Stellar smart contracts with AI-assisted workflows. Generate and deploy secure smart contracts using proven standards, guided by machine copilots.

Try Contracts MCP →

Open Source Tools

Relayer

Enable secure relaying for Stellar transactions. Anchors and apps can simplify backend operations and transaction flow management by using OpenZeppelin Relayer.

Docs → Github →

Monitor

Track Stellar applications in real time. Detect anomalies, log events, and trigger automated responses to safeguard Stellar smart contracts and protect users.

Docs → Github →

UI Builder

Generate front-ends for Stellar contracts in seconds. Configure functions and export a complete React app to interact with deployed contracts.

Try UI Builder →

Soroban Security Detectors SDK

Spot issues in Stellar contracts before they reach mainnet. This SDK enables developers to build custom static-analysis scanners and comes with prebuilt checks to catch common pitfalls early.

Github →
